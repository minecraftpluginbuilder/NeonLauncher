## Neon Launcher

A fast, stable, and open-source Minecraft launcher focused on performance and a clean user experience.

## Project Status

Neon Launcher is in early development.

## Features (Planned)

- Lightweight launcher architecture
- Multiple Minecraft version support
- Profile and account management
- Modpack-friendly workflow
- Fast startup and update flow
